The solution is to explicitly set the `type` attribute of the button element to `button` or another appropriate value.

<form action="submit.php" method="post"> <button type="button">Click Me</button> <input type="text" name="username"> <input type="submit" value="Submit"></form>

Or, you could use a different type of input for your submit button, avoiding the button element altogether. 

<form action="submit.php" method="post"> <input type="text" name="username"> <input type="submit" value="Submit"></form>

Using a separate submit input element provides more explicit control over the form submission process. In this case, a user will not experience double or unintended submissions because of clicking the button.
